How to make an ommlete, for robots

When asked by a user if a robot can make scrambled eggs, typically a robot does not contain the necessary code to execute such a task. That's the reality of many robots manufactured today, failing over and over and encountering an average of 6 errors comprehending how eggs work. This tutorial will cover in robot terms how to scramble an egg, from walking into the kitchen to serving it.

You will need the following items to produce scrambled eggs
  • 14g cooking oil
  • 1 to 2 eggs depending on user hunger
  • Cooking pan with handle

Instructions:

  1. Clean appendages via manufacturer recommended methods
    Warning
    Ensure cleaning process is food safe. If not food clean, return user invalid process error.
  2. Open refrigerator
  3. Retrieve eggs from refrigerator (or countertop if not mass produced)
  4. Close refrigerator
  5. Put pan on stove burner
  6. Turn on stove. There are seperate instructions depending on model of stovetop. Check user manual for stove if needed
  7. Put pan on stove, with concave side facing up and away from the burner
